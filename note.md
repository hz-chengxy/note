# GNU 规范
- 2.3.0
- 2: major 主版本号
- 3: minor 次版本号
- 0: patch 补丁，偶数是稳定版，基数是测试版

### npm 版本控制前缀
- npm view vue versions
- ^: 锁定主版本号
- ~: 锁定次版本号
- *: 固定版本
- 无：安装最高版本

# 物理像素 VS 逻辑像素

- 逻辑像素：我们开发人员在CSS里写的px
### DPR: devicePixelRatio
- 物理像素/逻辑像素
